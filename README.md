量刑系統 UI Redesign
====================

環境設定
--------

本專案以 `yarn` 套件管理程式打點環境設定。詳細安裝步驟，請參考 [Node.js](https://nodejs.org/en/download/package-manager/) 和 [`yarn`](https://yarnpkg.com/getting-started/install) 的官方文件。

```bash
npm install -g yarn
```

安裝專案的相依套件，啟動開發伺服器並在瀏覽器中打開：

```bash
yarn install
yarn start
```

編譯
----

將程式碼編譯成靜態頁面，輸出至 `build/`。

```bash
yarn build
```
